# TData

TData is a processor for files matching the original Tibia 7.7 data format.

## Commands

### NPC Responses

```
./tdata npc responses --dir /mnt/e/Tibia/src/ | head -n3
A Prisoner: Apples! Real apples! Man I love them! Can I have one? Oh please say yes!
A Prisoner: Better save time than comitting a crime. I am a poet and I know it!
A Prisoner: But first tell me your favourite colour please!
```
